voici le backend de notre projet gentlebaby pour le moment il n'y a qu'une partie mail .

## api mail & envoie d'invitation :
pour ma réfléxion j'ai fait quelque recherche mais en ne trouvant pas j'ai donc 
demander a l'ia mistral si il existait un moyen d'envoyer des mail avec 
express.js (l'api que j'utilise actuellement).

la réponse est qu'il n'existe pas de moyen avec express.js directement mais qu'il 
y a des moyens avec node.js (se qui reviens a la meme chose) via Nodemailer.


1. Installer Nodemailer :

```
npm install nodemailer
```
